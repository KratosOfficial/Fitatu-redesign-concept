<h1 align="center">How to Use 🤔</h1>

````
- Clone this repository:
$ https://github.com/KratosOfficial/Fitatu-redesign-concept

- Enter in directory:
$ cd fitatu

- For install dependencies:
$ flutter packages get

- Run the app: 
$ flutter run

````
